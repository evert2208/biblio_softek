# Gestión de Libros y Autores con .NET y SQLServer

## Descripción
Este proyecto esta desarrollado en **.NET** que permite gestionar libros y autores.  
Permite crear, consultar, actualizar y eliminar registros aplicando buenas prácticas de arquitectura, Entity Framework y manejo global y personalizado de excepciones con las especificaciones requeridas.
Se utilizo el maximo permitido de 2 quemada en el Api para realizar pruebas de las excepciones requeridas de la prueba. 

## Funcionalidades
- CRUD de Autores  
- CRUD de Libros    
- Manejo global y personalizados de excepciones 
- Migraciones con Entity Framework
- Endpoints con Swagger
